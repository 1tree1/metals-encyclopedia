순철 純鐵 — Pure Iron

C < 0.02% · BCC α-Ferrite
BCC (Body-Centered Cubic) 체심입방격자

결정구조

순철은 상온(α-Fe)에서 BCC(체심입방격자) 구조를 가집니다. 단위격자(unit cell)의 8개 꼭짓점과 1개 중심에 Fe 원자가 배치됩니다. 격자상수 a = 0.2866nm, 배위수 8, 충전율 68%입니다.

상변태

  • α-Fe (BCC) → 912°C → γ-Fe (FCC) → 1394°C → δ-Fe (BCC)
  • 768°C 이하: 강자성(페라이트)
  • 탄소 고용한도: α-Fe에서 최대 0.02%

연철 鍊鐵 — Wrought Iron

C < 0.08% · Ferrite + Slag Inclusions
Ferrite 기지 + 섬유상 슬래그 개재물

미세조직

연철은 페라이트(α-Fe) 기지 속에 규산염 슬래그(silicate slag) 개재물이 섬유 형태로 분포된 복합 구조입니다. 슬래그는 주로 FeO·SiO₂(파야라이트)로 구성됩니다.

제조법

  • 퍼들링(puddling) 공정: 수작업으로 교반
  • 반복 단조·접힘으로 슬래그 분포 → 결이 생김
  • 산업혁명 이전 주요 구조 재료

강철 鋼鐵 — Steel

C 0.02~2.11% · Ferrite + Pearlite (+ Cementite)
펄라이트 층상구조 (Ferrite + Cementite 교대 배열)

미세조직

강철의 핵심은 펄라이트(Pearlite)입니다. 727°C 공석반응에서 γ-Fe(오스테나이트)가 α-Fe(페라이트)와 Fe₃C(시멘타이트)의 층상구조(lamellar)로 분해됩니다. 밝은 층이 페라이트, 어두운 층이 시멘타이트입니다.

탄소 함량별 조직

  • 아공석강 (C < 0.77%): 초석 페라이트 + 펄라이트
  • 공석강 (C = 0.77%): 100% 펄라이트
  • 과공석강 (C > 0.77%): 펄라이트 + 초석 시멘타이트

주철 鑄鐵 — Cast Iron

C 2.11~6.67% · Pearlite/Ferrite + Graphite/Cementite
회주철: 편상흑연 / 구상흑연주철: 구상흑연

미세조직

주철에서 탄소는 두 가지 형태로 존재합니다: 흑연(Graphite) 또는 시멘타이트(Fe₃C). 냉각 속도와 Si 함량에 따라 흑연의 형상이 결정됩니다.

주철의 종류

  • 회주철(Gray): 편상흑연(flake graphite) — 진동 흡수 우수
  • 구상흑연주철(Ductile): Mg 첨가 → 구형 흑연 — 연성 확보
  • 백주철(White): 전체 시멘타이트 — 극경질·극취성
  • 가단주철(Malleable): 백주철 열처리 → 단괴상 흑연

흑연의 역할

흑연은 육방정계(HCP) 결정으로, 층간 반데르발스 결합이 약해 윤활 효과를 줍니다. 편상흑연은 균열의 시작점이 되어 취성을 유발하지만, 구상으로 만들면 응력 집중을 완화하여 강철에 가까운 기계적 성질을 얻습니다.
